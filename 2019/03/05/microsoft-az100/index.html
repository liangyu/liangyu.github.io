AZ-100 Azure Infrastructure and Deployment

2019.03.05 Liang Yu  热度 ℃

Test Preparation to being a full-fledged Microsoft Azure Administrator Certificate

AZ-100 Azure Infrastructure and Deployment

40-60 questions, exam will be around 180 minutes, pass with 700 marks

Manage Azure subscriptions and resources (15-20%)

Manage Azure subscriptions

1. Assign administrator permissions
2. Configure cost center quotas and tagging
3. Configure Azure subscription polices at Azure subscription level

Analyze resource utilization and consumption

1. Configure diagnostic settings on reources
2. Create baseline for resources
3. Create and test alerts
4. Analyze alerts across subscription
5. Analyze metrics across subscription
6. Create action groups
7. Monitor for unsused resources
8. Monitor spend
9. Report on spend
10. utilize Log Search query functions
11. view alerts in Log Analytics

Manage resource groups

1. Use Azure policies for resource groups
2. Configure resource locks
3. Configure resource policies
4. Implement and set tagging on resource groups
5. Move resources across resource groups
6. Remove resources groups

Implement and manage storage (20-25%)

Create and configure storage accounts

1. Configure network access to the storage account
2. Create and configure storage account
3. Generate shared access signature
4. Install and use Azure Storage Explorer
5. Manage access keys
6. Monitor activity log by using Log Analytics
7. Implement Azure storage replication

Import and export data to Azure

1. Create export from Azure job
2. Create import into Azure job
3. Use Azure Data Box
4. Configure and use Azure blob storage
5. Configure Azure content delivery network (CDN) endpoints

Configure Azure files

1. Create Azure file share
2. Create Azure File Sync service
3. Create Azure sync group
4. Troubleshoot Azure File Sync

Implement Azure backup

1. Configure and review backup reports
2. Perform backup operation
3. Create Recovery Services Vault
4. Create and configure backup policy
5. Perform a restore operation

Deploy and manage virtual machines (VMs) (20-25%)

Create and configure a VM for Windows and Linux

1. Configure high availabilty
2. Configure monitoring, networking, storage, and virtual machine size
3. deploy and configure scale sets

Automate deployment of VMs

1. Modify Azure Resource Manager(ARM) template
2. Configure location of new VMs
3. Configure VHD template
4. Deploy from template
5. Save a deployment as an ARM template
6. Deploy Windows and Linux VMs

Manage Azure VM

1. Add data discs
2. Add network interfaces
3. Automate configuration management by using PowerShell Desired State Configuration (DSC) and VM Agent by using custom script extensions
4. Manage VM sizes
5. Move VMs from one resource group to another
6. Redeploy VMs

Manage VM backups

1. Configure VM backup
2. Define backup policies
3. Implement backup policies
4. Perform VM restore

Configure and manage virtual networks (20-25%)

Create connectivity between virtual networks

1. Create and configure VNET peering
2. Create and configure VNET to VNET
3. Verify virtual network connectivity
4. Create virtual network gateway

Impement and manage virtual networking

1. Configure provate and public IP addresses
2. Network routes
3. Network interface, subnets, and virtual network

Configure name resolution

1. Configure Azure DNS
2. Configure custom DNS settings
3. Configure private and public DNS zones

Create and configure a Network Security Group (NSG)

1. Create security rules
2. associate NSG to a subnet or network interface
3. identify required ports
4. evaluate effective security rules

Manage identities (15-20%)

Manage Azure Active Directory (AD)

1. Add custom domains
2. Configure Azure AD Identity Protection
3. Azure AD Join, and Enterprise State Roaming
4. Configure self-service password reset
5. Implement conditional access policies
6. Manage multiple directories
7. Perform an access review

Manage Azure AD objects (usres, groups, and devices)

1. Create users and groups
2. Manage user and group properties
3. Manage device settings
4. Perform bulk user updates

Implement and manage hybrid identites

1. Install and configure Azure AD Connect
2. Configure federation and single sign-on
3. Manage Azure AD Connect
4. Manage password sync and writeback
