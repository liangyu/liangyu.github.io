Design Network Layer for iOS Application

Feb 7, 2018
  1. 同一个View Controller 调用不同的API 返回的数据结构不一致
  2. API 返回的数据不能直接被 View Controller 使用
  3. 需要保存一份返回的原始数据的copy，存在cache里。这样如果API有 calling的时间间隔限制，可以设置一个timer，超过这个timer以前，都直接显示cache里面的内容
