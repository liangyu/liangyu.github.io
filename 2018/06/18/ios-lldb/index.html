LLDB XCode 内置调试工具

2018.06.18 Liang Yu  热度 ℃

LLDB是个开源的内置于XCode的调试工具，常用的调试工具有：

Example:

1.打印值、修改值、调用方法

1.1 P 和 Po

// P 还能进行常量的进制转换 transform between different numeral system
// Decimal, Binary, Octal, Hexadecimal

  • 默认打印为10进制
    (lldb) p 100
    (int) $6 = 100

  • 转16进制
    (lldb) p/x 100
    (int) $7 = 0x00000064

  • 转8进制
    (lldb) p/o 100
    (int) $8 = 0144

  • 转二进制
    (lldb) p/t 100
    (int) $9 = 0b00000000000000000000000001100100

  • 字符转10进制数字
    (lldb) p/d ‘A’
    (char) $10 = 65

  • 10进制数字转字符
    (lldb) p/c 66
    (int) $11 = B\0\0\0

1.2 expression 修改参数值

  • 可以打印值
    (lldb) expression a
    (int) $0 = 1

  • 修改值
    (lldb) expression a = 10
    (int) $1 = 10
    // now a is 10, a + b = 12 instead of 3
    (lldb) p a
    (int) $2 = 10

1.3 call 方法调用
同样为输出值+值类型+引用名

2.Thread

2.1 堆栈打印 thread backtrace
(lldb) bt 10 (only output 10 lines)

2.2 thread return 跳出当前方法的执行
(lldb) thread return 999
–> 在断点停下后，直接让方法 func3 返回999

(lldb) thread return
–> 直接跳出方法，不让代码执行某个方法，
