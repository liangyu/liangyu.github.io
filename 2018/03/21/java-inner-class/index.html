2018.03.21 Liang Yu  热度 ℃

(regular) Inner Classes

  • event-handling class

Any kinds of inner classes, which are not belongs to the following three types. A regular
inner class is scoped inside another class’s curly braces, but outside any method code.

Inner class can access to member of the enclosing (outer) class, as if the inner class were part of the outer class.

The only way you can access the inner class is though a live instance of the outer class.

Instantiating an Inner Class

You must have an instance of the outer class to tie to the inner class.

Instantiating an Inner Class from Within the Outer Class

It works only if there is already an instance of the outer class – the instance running the
makeInner() method.

Referencing the Inner or Outer Instance from Within the Inner Class

review about the this

  1. the keyword this can be used only from within instance code. not within static code.
  2. the this reference is a reference to the currently executing object. In other words,
    the object whose reference It is was used to invoke the currently running method.
  3. The this reference is the way an object can pass a reference to itself to some other
    code, as a method argument:

Summary:

  1. To reference the inner class instance itself, from within the inner class code, use this.
  2. To reference the “outer this” (this outer class instance) from within the inner class code,
    use NameOfOuterClass.this (example, MyOuter.this)
