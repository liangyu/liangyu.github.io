Messaging

  1. Target - Operation 只限制一种事件，比如按下按钮，发生事件的一方会向制定的目标对象发送一个之前设定好的动作消息。针对不同的事件，需要创建不同动作消息。

  2. delegate
    一个委托可以发送多种事件消息

  3. NSNotificationCenter (处理App内部调对象通信)。通过它，对象可以注册感兴趣的系统事件。
    Low Memory –> UIApplicationDidReceiveMemoryWarningNotification
    Device Rotation –> UIDeviceOrientationDidChangeNotification
