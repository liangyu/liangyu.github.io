Networking Layers

Mar 3, 2017

Host Layer: #7 Application Data High-level APIs, including resource sharing, remote file access.
e.g. Telnet, FTP, SMTP

Host Layer: #6 Presentation Data Translation of data between a networking service and an application, including character
encoding, data compression and encryption/decryption.

DNS, RIP, SNMP

Host Layer: #5 Session Data Managing communication sessions, i.e. continous exchange of information in the form of
multiple back-and-forth transmissions between two nodes.

Host Layer: #4 Transport Segement (TCP) Reliable transmission of data segments between points on a network, including segmentation,
Datagram (UDP) acknowledgemenbt and multiplexing.

TCP, UDP

Host Layer: #3 Network Packet Structuring and managing a multi-node network, including addressing, routing and traffic control

IP, IGMP, ICMP, ARP

Host Layer: #2 Data link Frame Reliable transmission of data frames between two nodes connected by a physical layer

Ethernet, Token Ring, ATM

Host Layer: #1 Physical Bit Transmission and reception of raw bit streams over a physical medium
F
rame relay

At each level, two entities at the communicating devices exchange the protocol data units by means of a layer protocol. Each PDU contains a payload, called the SDU, along with protocol-related headers or footers.

  • PDU: product data unit
  • SDU: service data unit
  • HTTP: Hyper Text Transfer Protocol
  • FTP: File Transfer Protocol
  • SMTP: Simple Mail Transfer Protocal
  • DNS: Domain Name System – 将域名转化成 IP
  • RIP: Routing Information Protocol
  • SNMP: Simple Network Management Protocol
  • TCP: Transmission Control Protocol
  • UDP: User Datagram Protocol
  • ICMP: Internet Control Message Protocol
  • IGMP: Internet Group Message Protocol
  • ARP: Address Resolution Protocol
  • IP: Internet Protocol
