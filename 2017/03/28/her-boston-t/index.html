Version 1.2 includes bug fixes for HerBostonT. This update:

  • Fixes an issue where an extra letter 2

  • sometimes, MBTA server fails to provide the correct trip headsign information. Instead, the plain value “Trip Destination” is returned. Modified the App to parse this information to customer-friendly words as “Heading towards You”.

Version 1.2 also includes App feature:

  • Display instruction on the start page. Application enters into main view 3 seconds after user slide into the 3rd view.
